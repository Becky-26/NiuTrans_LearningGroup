# neu_nlplab_learning

本项目为小牛翻译实验室供同学和老师们学习交流的平台。内容主要包括论文、学习资料、个人或内部开发的代码、有用的开源项目等。同学们可以借助这个平台快速找到自己需要的学习资源，每个人也可以推荐分享有益的各种资源。希望大家能真正利用这个平台！
